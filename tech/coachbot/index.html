CoachBot

A Clean Language Coaching Robot

Clean

Using clever language processing, CoachBot uses the 12 original clean language questions to help you get to the root of your problem.

Private

CoachBot respects your privacy. It works completely offline and doesn't store or transmit any of your session information.

Open

CoachBot is completely free and open-source software. Built on Node.js's awesome framework, you can even include CoachBot in your own Javascript projects!

Helpful Commands

Command Action
!help Display help text
!version Display version info
!goodbye End the current session
!? Type this if the bot isn't making sense
!reset (client/settings/all) Reset the client/settings/both to default
!name name Change the client's name
!botName name Change the bot's name
!sound true/false Toggle sound effects
!debug true/false Toggle debug console logging
!logging Toggle verbose console logging mode
!capitalise true/false Toggle sentence case for bot replies
!quote Toggle quotation marks in bot's replies
!rv Toggle verb form flipping
